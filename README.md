# learn-git
学git，认识git，第一步要干啥？
> 值得一提的是，本教程适配mac系统，但是Windows系统暂时不适配，使用Windows电脑开发的同学可以下载一个gist
既然我们课行校园通使用git来管理代码及发布版本，研发同学们就需要了解git最基本的使用方法，这包括但不限于：
1. 克隆仓库；
2. 创建本地分支；
3. 添加本地修改；
4. 写commit信息；
5. 提交修改到远程仓库；
